# File Map

File ownership map for `H5API/`.

- `H5API/app.py`
  - Flask app + route handlers + storage/prefix utilities.

- `H5API/README.md`

- `H5API/.env`
  - local runtime configuration.

- `H5API/requirements.txt`
  - dependencies (`flask`, `flask-cors`, `python-dotenv`).

- `H5API/.gitignore`
  - local ignore rules.
